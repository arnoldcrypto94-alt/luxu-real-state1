# Technology Stack: LuxeEstate

This document outlines the primary technologies, frameworks, and libraries used to build the LuxeEstate platform.

## Frontend

*   **Language:** [TypeScript](https://www.typescriptlang.org/)
    *   *Description:* A statically typed superset of JavaScript that adds type safety to the application.
*   **Framework:** [Next.js 16.2.0](https://nextjs.org/)
    *   *Description:* The core React framework providing server-side rendering, static site generation, and a robust routing system (App Router).
*   **UI Library:** [React 19.2.4](https://react.dev/)
    *   *Description:* A JavaScript library for building user interfaces, used as the foundation for all components.
*   **Styling:** [Tailwind CSS 4.0](https://tailwindcss.com/)
    *   *Description:* A utility-first CSS framework used for all styling. It allows for rapid UI development directly within the markup.

## Development & Tooling

*   **Linting:** [ESLint](https://eslint.org/)
    *   *Description:* Used to find and fix problems in the TypeScript/JavaScript code, ensuring code quality and consistency.
*   **Package Manager:** [npm](https://www.npmjs.com/)
    *   *Description:* Used for managing project dependencies and running scripts.
